Suggested: drop the DiskSink `slab_` member and compute `slab_bytes_` from a local, or keep only one of the two and derive the other at the single use site.

Not removable without an interface change, but worth a debug assert that `slab_elems == slab_` to catch a caller mismatch rather than discarding it. Suggested: replace the `(void)` discard with a debug-only `assert(slab_elems == slab_)` so the silently-ignored argument cannot mask a caller bug.
